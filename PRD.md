# Product Requirements Document (PRD)

## Product

svglogo.dev

## Product Type

Open-source browser-based logo generator.

## Objective

Provide a fast and minimal tool that allows users to generate a logo by selecting an icon, customizing its color and background, and exporting the result in common formats. The tool should feel lightweight, immediate, and visual, focusing on speed rather than complex design workflows.

The initial version intentionally limits scope to icon-based logos without typography.

## Icon Selection

Users must be able to:

* open an icon selection dialog
* search icons using keywords
* preview icons
* select an icon

The selected icon becomes the main graphic on the canvas.

---

## Icon Customization

Users must be able to adjust:

* icon color
* icon size (optional but recommended)

All updates must reflect instantly on the canvas.

## Export

Users must be able to export the canvas as:

SVG
PNG
ICO

---

# Non-Goals for MVP

The first version will **not include**:

* text or typography
* brand name generation
* AI logo suggestions
* advanced layout systems
* multi-layer editing
* saving projects
* user accounts

These may be considered for later versions.
