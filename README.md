## ⚙️ How svglogo Works

svglogo loads icons from multiple popular icon sets. You pick icons and adjust their size and color. The app combines these into one logo design. It then converts the image into SVG, PNG, or ICO format.

The app uses Vite for fast loading and Tailwind CSS for styling. The interface is simple enough for users without design experience.
